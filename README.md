# Maingraph for MC

![一个徽标，显示“ModLoader”是“NeoForge”](https://img.shields.io/badge/ModLoader-NeoForge-orange.svg)


**Maingraph for MC** 是一个为 Minecraft 设计的可视化蓝图引擎。它通过直观的节点连接，无需编写（或者很少编写）任何代码以实现自定义逻辑。

---

## 快速上手

1. **安装**: 将 mod 放入你的 Minecraft `mods` 文件夹（需 NeoForge）。
2. **打开编辑器**: 进入游戏后，按下 `Ctrl + M` 键打开蓝图管理器。
3. **创建蓝图**: 输入名称并点击“创建”。
4. **开始编写**: 右键点击画布添加节点，连接白色箭头（执行流）和彩色圆点（数据流）。
5. **保存运行**: 点击“保存”，你的逻辑就已经在后台开始运行了。

详细教程请参考 [mc.maingraph.opens.ltd/docs-cn/首页](https://mc.maingraph.opens.ltd/docs-cn/首页)。
