# 获取列表长度 (List Length)

- **分类**: 变量 -> 列表
- **内部 ID**: `list_length`

## 节点功能
统计并返回指定列表中包含的项目总数。

![获取列表长度节点](https://free.picui.cn/free/2026/01/03/69584debdec99.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表 | `list` | 列表 (LIST) | 需要测量长度的列表。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 长度 | `length` | 浮点数 (FLOAT) | 列表中的项目总数。 |

## 逻辑规则
1. **空值统计**：该节点会统计所有由 `|` 分隔的部分。如果列表字符串中存在连续的 `|`（代表空项），这些空项也会被计入长度。
2. **返回类型**：虽然长度是整数，但为了兼容数学计算节点，该输出值会以浮点数格式提供。
3. **空列表**：完全没有内容的列表将返回 `0`。
