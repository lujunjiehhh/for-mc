# 添加到列表 (List Add)

- **分类**: 变量 -> 列表
- **内部 ID**: `list_add`

## 节点功能
向现有列表的末尾添加一个新项目。

![添加到列表节点](https://free.picui.cn/free/2026/01/03/69584de939da8.png)

## 入参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表输入 | `list_in` | 列表 (LIST) | 原始列表。 |
| 项目 | `item` | 字符串 (STRING) | 要添加的内容。 |

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 列表输出 | `list_out` | 列表 (LIST) | 添加新项目后的完整列表。 |

## 逻辑规则
1. **自动初始化**：如果输入的原始列表为空，输出将直接是待添加的项目。
2. **特殊字符转义**：如果添加的项目内容中包含 `|` 符号（列表的内部标识符），节点会自动为该项目包裹双引号 `""`，以防止列表结构损坏。
3. **追加方式**：新项目总是被添加到列表的最末尾。
