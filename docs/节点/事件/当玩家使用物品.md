# 当玩家使用物品

- **分类**: 事件
- **内部 ID**: `on_use_item`

## 节点功能
当玩家使用物品（如右键吃东西、使用工具等）时触发。输出物品 ID 和该玩家的 UUID。

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 使用物品后的逻辑执行出口。 |
| 物品 ID | `item_id` | 字符串 (STRING) | 所使用物品的注册名（如 `minecraft:apple`）。 |
| UUID | `uuid` | 唯一标识符 (UUID) | 使用物品的玩家的唯一标识符。 |
