# 当玩家敲击时

- **分类**: 事件
- **内部 ID**: `on_player_attack`

## 节点功能
当玩家攻击（左键敲击）另一个实体时触发。输出被攻击者的 UUID 和攻击者（玩家）的 UUID。

## 入参
无入参

## 出参
| 端口名称 | 内部 ID | 类型 | 说明 |
| :--- | :--- | :--- | :--- |
| 输出 | `exec` | 执行流 (EXEC) | 玩家攻击后的逻辑执行出口。 |
| 被攻击者 UUID | `victim_uuid` | 唯一标识符 (UUID) | 被攻击实体的唯一标识符。 |
| UUID | `uuid` | 唯一标识符 (UUID) | 攻击者（玩家）的唯一标识符。 |
